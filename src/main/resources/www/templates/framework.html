What does it do?

The Synload Framework is a backend server for any kind of project you are working on. The server implements the Jetty webserver as well as the Jetty websocket server. There are two seperate routing methods that the framework uses. One is event driven, which is multithreaded for the websocket requests and then the response is added to a queue. The second method is registering HTTP requests by the URI, the HTTP can also be routed using the event handler.

Modules, what are they?

Modules are what make Synload Framework unique. Dragging and dropping modules into the mods directory and restarting the server will give your site life. You can create your own modules and multiple teams can work on seperate aspects of the site using multiple modules. You can make anything from a video converting control panel to a game server.
